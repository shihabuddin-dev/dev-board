Question-1: What are the different ways to select an element in the DOM?

You can select elements in the DOM using getElementById, querySelector, and querySelectorAll. querySelector is the most flexible as it works with IDs, classes, and attributes like CSS selectors.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where instead of adding event listeners to multiple child elements, you add a single listener to their parent. It uses event bubbling to handle events efficiently.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and moves upward through its parent elements in the DOM. It allows handling child events from a parent element efficiently.
